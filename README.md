# 🏦 Dashboard - Monitoreo Financiero
 
## Descripción
Dashboard desarrollado en Power BI que automatiza el monitoreo financiero de una institución bancaria, integrando cinco fuentes de datos transaccionales en una única vista ejecutiva que incluye rentabilidad, liquidez, solvencia y calidad de cartera.
 
---
 
## Objetivo
Eliminar la dependencia de reportes manuales y consolidaciones tardías, poniendo en manos del director una vista ejecutiva del estado real del banco al cierre de cada período, con comparación automática con el mes anterior y alertas visuales de desviación, para responder en segundos: ¿cómo está el banco hoy versus el mes anterior?

## Características
- 8 KPIs estratégicos (ROE, ROA, NIM, liquidez, solvencia, mora, cobertura, costo de riesgo)
- Comparación automática mes a mes
- Sistema de alertas visuales por color ante desviaciones
- Visualizaciones de tendencia por período
- Consolidación de 5 fuentes de datos transaccionales
- Vista ejecutiva de rentabilidad, liquidez, solvencia y calidad de cartera
---
 
## Tecnologías
- Power BI Desktop
- DAX
- SQL Server
- Power Query (M)
---
 
## Dataset
**Fuente:** Datos sintéticos generados para fines académicos y de portafolio, con una estructura de modelo que replica estándares reales de información financiera regulatoria en Ecuador (SEPS / SBS).
